objective-c 格式化输出

所有的类都是NSObject的子类
OC中没有包的概念，OC中只有四种简单数据类型
int、float、double和char

扩展类型:
long、long long、short、unsigned和signed

占位符：

  1. char %c
  2. short int %hi %hx %ho
  3. unsigned short int %hu %hx %ho
  4. int %i %x %o
  5. unsigned int %u %x %o
  6. long int %li %lx %lo
  7. unsigned long int %lu %lx %lo
  8. long long int %lli %llx &llo
  9. unsigned long long int %llu %llx %llo
  10. float %f %e %g %a
  11. double %f %e %g %a
  12. long double %Lf $Le %Lg
  13. id 指针（这里间接说明id是一个指向objc_object结构体的指针） %p
  14. bool %d 0表真，1表假
